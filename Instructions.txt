1. Unzip all files into a new directory. 
2. Open that directory in terminal.
3. run: python -m venv env
4. run: .\env\Scripts\activate
5. run: pip install -r requirements.txt
6. Comment out line 13 in setup.py if you want to see the browser window.
7. run: python analyse.py
8. A screenshot is generated by the name of checkpoint2.png
